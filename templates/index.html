⚠ Fix: Alpaca showing $0 balance
Your paper account has separate API keys from your live account. The $0 means your .env has your live keys but is hitting the paper endpoint — or vice versa.

Fix: Go to app.alpaca.markets → top-left dropdown → switch to "Paper Trading" → API Keys → generate keys from that screen. Those keys will show your $1M paper balance.
Alpaca: Paper → Real Money
1
Open live Alpaca account at app.alpaca.markets → fund via ACH bank transfer → generate Live API keys
Takes 1–3 days to settle. Minimum $1 — no minimum required.
2
Update .env with live keys → Settings → change mode to Live → Save
3
Test with $10 → Deploy $10 → verify trade appears in Alpaca dashboard → scale up

NDAX Crypto Connection
NDAX has a full WebSocket API — WealthPilot connects via the ccxt library which handles the protocol automatically.
1
Install ccxt:
pip install ccxt
2
Get NDAX API keys: ndax.io → account → API Keys → generate key + secret + note your User ID
3
Add to .env:
NDAX_API_KEY=your_key_here
NDAX_API_SECRET=your_secret_here
NDAX_USER_ID=your_user_id_here
4
Restart app → Crypto tab will show live NDAX positions and activate auto-exit logic
